Инструкции по использованию Postman коллекции
- У вас должен быть аккаунт на GitHub, если его нет, создайте его.
- Создайте репозиторий на GitHub.
- Создайте API-ключ на GitHub: https://github.com/settings/tokens
- Откройте коллекцию в Postmam
- Откройте документацию по адресу: https://docs.github.com/ru/rest/issues/issues?apiVersion=2022-11-28.
- Найдите пункты: 1 - Create an issue; 2 - List repository issues; 3 - Update an issue; 4 - Unlock an issue.
- Замените в коллекции значения соответственно: owner-владелуц репозитория, repo-репозиторий, your-token-ваш токен, assignees-ваш логин; на ваши реальные данные.
- Запустите коллекцию.
